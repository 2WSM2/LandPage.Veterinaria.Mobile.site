# 🐾 Site Clínica Veterinária - Dra. Thayssa Galvão

Landing page desenvolvida para clínica veterinária, com foco em agendamento de consultas via WhatsApp, interface responsiva e experiência do usuário (UX) humanizada.

## 🚀 Tecnologias Utilizadas
- **HTML5**: Semântica e estrutura.
- **CSS3**: Flexbox, design responsivo e animações.
- **JavaScript**: Lógica de formulário, slideshow automático e scroll suave.
- **FontAwesome**: Ícones para interface.

## 🎯 Funcionalidades
- [x] Formulário de contato com envio direto para WhatsApp.
- [x] Slideshow de imagens no banner principal.
- [x] Design mobile-first (adaptado para celulares).
- [x] Seção de serviços com grid responsivo.
